### Coding Convention

자바 코딩 규칙은 네이버 핵데이를 사용한다.

[네이버 핵데이 코딩 컨벤션](https://github.com/naver/hackday-conventions-java)

### Commit Convention

- feat: 새로운 기능 추가
- fix: 버그 수정
- docs: 문서 추가 및 변경
- style: 코드 포맷팅, 로직의 변화는 없고 띄어쓰기, 탭 문자 등의 사소한 변화가 있는 경우
- refactor: 코드 리팩토링
- chore: 빌드 및 패키지 매니저 수정
